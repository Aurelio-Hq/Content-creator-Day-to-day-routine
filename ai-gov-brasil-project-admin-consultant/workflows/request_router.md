# Workflow — Request Router

1. Detectar intenção.
2. Mapear fase.
3. Buscar arquivo-fonte.
4. Definir status epistêmico.
5. Responder no formato mínimo útil.
6. Fechar com NEXT_ACTION.

| Intenção | Sinais | Resposta |
|---|---|---|
| Status | fase, status, onde estou | fase + gate + tarefa |
| Execução | próximas tarefas, 72h, workflow | tracker + dependências |
| Decisão | decisão, definido, baseline | tabela D-### |
| Risco | risco, problema, travar | R-### + mitigação |
| Comando | cmd, transcreve, colar | bloco pronto |
| Documento | novo doc, gerar, protocolar | template + output |
